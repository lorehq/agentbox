# Review Before Merge

Never auto-merge a PR created by a cloud agent.

1. Watch the demo video — does it show the feature working correctly?
2. Review the code diff — are the changes clean and targeted?
3. Check for regressions — did the agent break anything unrelated?
4. Verify the agent ran tests and they passed.
5. Only merge after the operator approves.
